# Vitalis AI

  Asistente personal de salud, fitness y estética con IA avanzada.

  ## Características

  - **Dashboard**: Progreso diario con anillo de progreso, macros y pasos
  - **Chat IA**: Asistente de salud y fitness con respuestas inteligentes
  - **Entrenamiento**: Tracker de ejercicios con checklist interactivo
  - **Nutrición**: Registro de comidas con desglose de macronutrientes
  - **Progreso**: Gráficas de peso corporal y récords personales
  - **Perfil**: Configuración personal y datos del usuario

  ## Tecnologías

  - HTML5 / CSS3 / JavaScript (Vanilla)
  - Progressive Web App (PWA) con Service Worker
  - Chart.js para gráficas
  - Font Awesome para iconos
  - Tema oscuro, idioma español

  ## Instalación

  ```bash
  npm install
  npm start
  ```

  La aplicación se sirve en `http://localhost:5000`

  ## PWA

  La app es instalable como PWA en dispositivos móviles y de escritorio.
  Soporta modo offline con estrategias de caché inteligentes.
